# juliasaboya.github.io

Portfolio profissional em HTML5, CSS3 e JavaScript puro, preparado para hospedagem no GitHub Pages.

## Estrutura

A base inicial foi organizada para separar rotas, assets, dados e componentes reutilizáveis, mantendo espaço para dezenas de projetos e artigos sem concentrar código em poucos arquivos.

Em `projetos/`, a organização inicial parte por categoria de conteúdo, não por projeto individual.

Referência rápida da arquitetura: [docs/architecture.md](docs/architecture.md)
